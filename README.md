# 互联网学历测试

> 21 种互联网学历人格，测测你在学历鄙视链里到底是哪一挂。

`互联网学历测试` 是一个面向中文互联网年轻用户的娱乐型人格测试项目。  
它不是严肃心理测评，而是把校园、升学、考研、就业、学历焦虑和网络梗做成一套可以测试、可以分享、也可以做人格图鉴卡的结果页。

项目内容风格参考 SBTI：

- 有梗
- 轻损
- 像被看穿
- 适合截图发群聊和朋友圈

## 项目特点

- **30 道场景题**：覆盖上课、自习、考试、宿舍、社团、升学、就业、学历包装等场景
- **15 个维度**：从学习能力、考试能力、学习态度、社交能力、生存策略五个模型做向量匹配
- **21 种互联网学历人格**：卷王、天才、小镇做题家、双非战神、专升本逆袭怪、精神本科生等
- **SBTI 风格结果页**：人格名、钩子文案、短描述、标签、匹配度
- **人格图鉴面板**：支持小图预览、分类标签、关键词搜索和筛选浏览
- **纯前端实现**：无后端依赖，适合静态部署

当前产品对外统一名称为 `互联网学历测试`，仓库目录与包名仍沿用 `hulianwangxueli-bti` 以保持部署地址和仓库链接稳定。

## 适合什么场景

- 做一个可传播的趣味测试页
- 做人格图鉴、分享海报、角色设定
- 做 AI 出图项目的人格基础设定
- 做校园梗 / 学历梗主题的小型互动产品

## 免费部署

### 方案一：Cloudflare Pages

这是最推荐的免费方案，原因是：

- 静态站支持非常稳
- 连 GitHub 后每次 push 自动发布
- 自带免费 HTTPS
- 后续绑定自定义域名最方便

操作方式：

1. 把项目推到 GitHub
2. 登录 Cloudflare Pages
3. 选择 `Connect to Git`
4. 选择你的仓库
5. 构建命令填写：`npm run build`
6. 输出目录填写：`dist`
7. 点击部署

### 方案二：GitHub Pages

项目已经内置了 GitHub Pages 的自动部署工作流。  
你只需要把仓库推到 GitHub，并在仓库里开启 Pages。

操作方式：

1. 把项目推到 GitHub
2. 打开仓库 `Settings -> Pages`
3. `Build and deployment` 选择 `GitHub Actions`
4. 保持仓库默认分支正常 push
5. 工作流会自动构建 `dist/` 并发布

### 关于国内访问

如果你希望“别人不开加速器也尽量能打开”，推荐优先使用：

- `Cloudflare Pages + 自定义域名`

原因是默认的 `github.io` 或平台二级域名在不同地区的访问稳定性会有波动，而自定义域名更适合后续做 DNS 和访问优化。  
这不能保证所有网络环境都完全稳定，但这是免费方案里相对更实用的路径。

## 21 种互联网学历人格

### 学业高压系

1. **卷王** - 别人熄灯了，他才刚热身
2. **天才** - 看着没学，其实都学了
3. **偏科怪** - 一科封神，一科送命
4. **小镇做题家** - 会做题，但人生不只考题

### 学渣摆烂系

5. **废物** - 不学，不装，也不挣扎
6. **摆烂王** - 能躺着，绝不坐起来
7. **蒙题高手** - 不会做，但很会赌
8. **钞能力者** - 分数不够，路子来凑
9. **作弊大师** - 不会做题，但会做局
10. **破防怪** - 成绩一出，先骂世界

### 中段生存系 / 校园关系系

11. **及格战士** - 不求优秀，只求活着
12. **抱佛脚** - 平时查无此人，考前突然显灵
13. **学生会干部** - 题做得一般，官话说得很满

### 升学路径系

14. **复读战神** - 别人高考结束，他高考重开
15. **双非战神** - 能力还没展示，学校先被审判
16. **专升本逆袭怪** - 不是原装本科，但硬是爬上来了
17. **野鸡院校受害者** - 校名很响，背景很虚

### 现实社会系 / 高端赛道系

18. **厂狗预备役** - 别人谈梦想，他先看包不包住
19. **精神本科生** - 学校一般，气势顶配
20. **中专文豪** - 书没读懂，人生先看透
21. **竞赛老嗨** - 普通考试，不配入眼

## 评分机制

当前实现采用 `15 维度向量 + 曼哈顿距离匹配`：

1. 每题选项得分为 `1 / 2 / 3`
2. 同一维度共 2 题，汇总为 `2 - 6 分`
3. 分值映射为 `L / M / H`
4. 与 21 种人格模式向量做距离匹配
5. 返回距离最近的人格结果

这种方式的好处是：

- 适合人格边界模糊的娱乐测试
- 能稳定跑出 21 种结果
- 后续方便增加“你也有点像谁”的扩展玩法
