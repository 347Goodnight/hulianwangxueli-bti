# 互联网学历测试

> 21 种互联网学历人格，测测你在学历鄙视链里到底是哪一挂。

`互联网学历测试` 是一个面向中文互联网年轻用户的娱乐型人格测试项目。  
它不是严肃心理测评，而是把学校层级、学位进阶、学术头衔、学历包装和网络梗做成一套可以测试、可以分享、也可以做人格图鉴卡的结果页。

项目内容风格参考 SBTI：

- 有梗
- 轻损
- 像被看穿
- 适合截图发群聊和朋友圈

## 项目特点

- **30 道场景题**：覆盖学校层级、学位升级、学历包装、升学焦虑和学术头衔想象等场景
- **15 个维度**：从学习能力、考试能力、学习态度、社交能力、生存策略五个模型做向量匹配
- **21 种互联网学历人格**：985圣体、职高战士、中专文豪、博士、本硕博直通车、长江学者、院士、诺奖得主等
- **SBTI 风格结果页**：人格名、头衔档案卡、短描述、标签、匹配度
- **人格图鉴面板**：支持小图预览、分类标签、关键词搜索和筛选浏览
- **纯前端实现**：无后端依赖，适合静态部署

当前产品对外统一名称为 `互联网学历测试`，仓库目录与包名仍沿用 `hulianwangxueli-bti` 以保持部署地址和仓库链接稳定。

## 适合什么场景

- 做一个可传播的趣味测试页
- 做人格图鉴、分享海报、角色设定
- 做 AI 出图项目的人格基础设定
- 做校园梗 / 学历梗主题的小型互动产品

## 免费部署

### 方案一：Cloudflare Pages

这是最推荐的免费方案，原因是：

- 静态站支持非常稳
- 连 GitHub 后每次 push 自动发布
- 自带免费 HTTPS
- 后续绑定自定义域名最方便

操作方式：

1. 把项目推到 GitHub
2. 登录 Cloudflare Pages
3. 选择 `Connect to Git`
4. 选择你的仓库
5. 构建命令填写：`npm run build`
6. 输出目录填写：`dist`
7. 点击部署

### 方案二：GitHub Pages

项目已经内置了 GitHub Pages 的自动部署工作流。  
你只需要把仓库推到 GitHub，并在仓库里开启 Pages。

操作方式：

1. 把项目推到 GitHub
2. 打开仓库 `Settings -> Pages`
3. `Build and deployment` 选择 `GitHub Actions`
4. 保持仓库默认分支正常 push
5. 工作流会自动构建 `dist/` 并发布

### 关于国内访问

如果你希望“别人不开加速器也尽量能打开”，推荐优先使用：

- `Cloudflare Pages + 自定义域名`

原因是默认的 `github.io` 或平台二级域名在不同地区的访问稳定性会有波动，而自定义域名更适合后续做 DNS 和访问优化。  
这不能保证所有网络环境都完全稳定，但这是免费方案里相对更实用的路径。

## 21 种互联网学历人格

### 名校光环系

1. **985圣体** - 还没张嘴，学校名就已经替他说完了
2. **211守门员** - 够不上顶配神校，但也不是谁都能随便踩的
3. **一本遗老** - 双一流都讲几年了，他还在怀念当年的一本线

### 学历落差系

4. **二本刺客** - 学校名一报出来，空气会先安静半秒
5. **精神本科生** - 学历未必够硬，气势必须先拉满
6. **专升本逆袭怪** - 不是原装本科，但硬是把车补上了
7. **大专战狼** - 学历可以矮一头，嘴上绝对不能输半句
8. **中专文豪** - 书没读明白，人生感悟倒先写了三大段
9. **职高战士** - 分流那天像被提前判了线，后来才发现社会根本不按统招卷子出题
10. **成考补票哥** - 白天上班晚上刷题，主打一个人生进度条自己补

### 学历包装系

11. **留学水硕** - 专业学没学明白不好说，英文校名肯定背熟了
12. **民办贵族** - 学费交得很响，学历落地时却还是轻飘飘
13. **野鸡院校受害者** - 校名听着像联合国，查起来像写字楼二层
14. **文盲** - 学历栏看着像空的，认知栏也没好到哪去

### 学位进阶系

15. **硕士保底哥** - 本科不够看，那就顺手再把硕士补到手
16. **博士** - 别人还在读书，他已经开始把头发读没了
17. **本硕博直通车** - 别人靠考试补票，他的人生像被系统直接续费

### 学术神坛系

18. **长江学者** - 别人还在读文献，他已经快被写进文献里了
19. **院士** - 学历体系写到这里，基本已经快到结尾彩蛋了
20. **诺奖得主** - 普通人写简历，他这种一般直接进教材
21. **学阀世家** - 别人拼学历是逆天改命，他拼学历像家族日常维护

## 评分机制

当前实现采用 `15 维度向量 + 曼哈顿距离匹配`：

1. 每题选项得分为 `1 / 2 / 3`
2. 同一维度共 2 题，汇总为 `2 - 6 分`
3. 分值映射为 `L / M / H`
4. 与 21 种人格模式向量做距离匹配
5. 返回距离最近的人格结果

这种方式的好处是：

- 适合人格边界模糊的娱乐测试
- 能稳定跑出 21 种结果
- 后续方便增加“你也有点像谁”的扩展玩法
